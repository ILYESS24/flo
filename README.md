# Flo AI API

API backend pour Aurora AI Studio.

## Déploiement

Le Dockerfile est configuré pour un déploiement simple. Tous les fichiers nécessaires sont à la racine du repository.

## Structure

```
.
├── api.py              # Point d'entrée de l'API
├── aurora_ai/          # Module principal
├── requirements.txt    # Dépendances Python
└── Dockerfile         # Configuration Docker
```
